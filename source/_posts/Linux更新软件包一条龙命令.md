apt-get update 更新索引

```bash
apt-get update
```

apt-get upgrade 更新软件

```bash
apt-get upgrade
```

apt-get dist-upgrade 升级

```bash
apt-get dist-upgrade
```

apt-get clean 删除缓存包

```bash
apt-get clean
```

apt-get autoclean 删除未安装的 deb 包

```bash
apt-get autoclean
```
